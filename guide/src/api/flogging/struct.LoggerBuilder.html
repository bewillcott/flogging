pub fn remove_file(self, filename: &str) -> Self

Remove an existing log file.

The purpose of this, is to allow resetting of the log file, each time a test run is done.

ยงNote

This must be called before adding the corresponding file handler that is going to open this file.

ยงParameters
  • filename - The name of the output log file. Must include any relevant path (relative or absolute).
ยงExamples

use flogging::*;

    .remove_file("mylog.txt")
    .add_file_handler("mylog.txt")
    .build();
